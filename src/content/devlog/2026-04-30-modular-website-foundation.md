The Kelaris website is moving from repeated static pages toward a modular foundation.

This update adds a structure that can support reusable layouts, shared navigation, content-driven pages, and build validation. The goal is to make the website easier to grow without losing the current visual identity.

The look remains the same direction: dark space atmosphere, glass panels, blue and teal accents, and the core public pitch:

**Build the machine. Hold the line.**

This foundation prepares the site for the next layer of work:

- Markdown-powered devlogs.
- A real media gallery.
- A press kit.
- A public roadmap page.
- Member and tester pages.
- Safer automated checks on every pull request.
